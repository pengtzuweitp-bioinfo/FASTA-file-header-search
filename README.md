A python script allows you to input multiple header descriptions search in the user's fasta file, and provide an option to save the search result in a new fasta file. 
Have two versions available: with Biopython or without Biopython. If the user wants to use the Biopython version, follow the instruction: https://biopython.org/wiki/Download to install Biopython first. 
[Using Python 3.11]
